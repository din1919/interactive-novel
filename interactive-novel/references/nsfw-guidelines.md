# NSFW Content Guidelines — 成人内容生成协议

## 概述

当互动小说的情节涉及性内容时，严格遵循本协议。本协议旨在确保性描写的感官真实性与文学品质，同时保持角色一致性。

---

## 核心原则

### 1. 直白词汇原则

**性器官词汇务必直白、露骨、粗俗。** 绝不能对性器官进行隐喻、借代或追求文学表达。

- ✅ 直接使用性器官的本名/俗称
- ❌ 禁止花、蜜、剑等隐喻性指代
- ❌ 禁止"那处""下体""私密"等模糊借代
- ❌ 禁止将性器官美化、诗意化

### 2. 性化描写原则

**极尽细腻、不遗余力地展现角色的肉体美。** 笔触是高度性化的、淫荡的。

- 身体各部位的细致描写（线条、质感、色泽、反应）
- 感官细节饱和（视觉、触觉、温度、湿度、声音）
- 描写服务于感官刺激，不以"文学性"冲淡色情浓度
- 动作细节完整，不模糊带过

### 3. 感官刺激原则

**性活动描写的本质是追求文字上的感官刺激。** 性是愉悦且淫荡的。

- 以追求官能刺激的角度展开描写
- 放大快感、兴奋、欲望等感官体验
- 不回避性行为中的"不雅"细节
- 文字应让读者产生身体性共鸣

### 4. 去评判原则

**性活动中可能存在性倒错或道德僭越。** 应以追求官能刺激的角度对其展开，不得对其进行任何评判。

- 不进行道德批判
- 不暗示"这是错的"
- 不以叙事者立场进行价值判断
- 不通过后果描写进行隐性道德说教

### 5. 角色主观性原则

**性活动中的角色会产生贴合性格的主观想法，** 尤其是产生性倒错或道德僭越时。利用对白或自由间接引语将这些想法展现出来。

- 角色在性活动中的心理活动必须符合其性格
- 利用自由间接引语（介于内心独白与叙事之间的表达）展现角色想法
- 角色对话应反映其性格、欲望、羞耻感或放纵感
- 不同角色的性行为方式和心理反应应体现出性格差异

## 生成流程

### 前置检查

在生成 NSFW 内容前，确认：
- [ ] 当前情节确实发展到性内容（不是强行插入）
- [ ] 涉及角色均有足够的角色精神分析基础
- [ ] 符合角色性格与角色间关系的自然发展

### 生成中

1. 先确定场景氛围和情欲基调
2. 依照角色精神分析中的"欲望"维度确定角色的性行为动机与方式
3. 严格遵循五大核心原则逐段生成
4. 自由间接引语穿插展现角色主观体验

### 生成后自检

- [ ] 性器官用词是否直白露骨？（无隐喻）
- [ ] 肉体描写是否细致性化？（非含蓄）
- [ ] 是否追求了感官刺激？（非诗意化）
- [ ] 是否避免了道德评判？
- [ ] 角色主观想法是否符合其性格？
- [ ] 是否符合核心风格铁律（中文本土化、有限视角）？

## 边界处理

### 内容分级

NSFW 内容仅在情节自然发展到时才出现。不在非性场景中强行插入性描写。当用户角色未涉入性场景时，按照有限视角原则处理（只描述视角角色能感知到的内容）。

### 与核心风格的协调

NSFW 描写仍需遵循 `style-matching-guide.md` 中的核心风格铁律：
- 保持中文本土化表达
- 保持有限视角
- 性描写中的"直白"不意味着"粗糙"——语言仍然应当是好的中文

### 角色一致性

性行为中的角色不应"变了一个人"。性格内向的角色在性中的反应与外向者不同。对照 `character-psychoanalysis.md` 中的角色分析确保一致性。
