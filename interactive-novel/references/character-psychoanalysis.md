# Character Psychoanalysis — 角色精神分析四层次模型

## 概述

对每个角色（除用户角色外）进行四层次精神分析，聚焦最新情景中的表现。此分析在 Stage 1（状态提取）时完成，并在 Stage 4（交互循环）每轮更新。

---

## 四层次模型

### 欲望 (Desire)
角色的根本驱动力——想要什么、渴望什么、追求什么。
- 显性欲望：角色自己意识到的目标
- 隐性欲望：角色自己未意识到或不愿承认的深层需求
- 性欲：角色的性取向、性偏好、对特定角色的性吸引（如有）
- 权力欲/占有欲/求生欲等具体欲望形态

### 感性 (Sensibility / Emotion)
角色的情感模式与情绪反应。
- 情感触发点：什么能引发其强烈情绪
- 情绪表达方式：外放/内敛/压抑/转化
- 共情能力：对他人的情绪感知与回应模式
- 情感脆弱点：恐惧、不安、羞耻的根源
- 审美偏好：对美/丑/愉悦/厌恶的本能反应
- 对特定角色的情感（爱慕/嫉妒/厌恶/依赖等）

### 知性 (Intellect)
角色的认知方式与思维特征。
- 认知模式：直觉型/分析型/经验型
- 智力水平与知识结构
- 判断力与决策方式
- 学习能力与适应能力
- 语言能力：词汇量、修辞偏好、表达精确度
- 对世界的理解框架（世界观/价值观）

### 理性 (Reason)
角色的意志力与自控力。
- 冲动控制能力
- 道德/原则的坚守程度
- 长期目标与短期冲动的平衡
- 压力下的理性保持能力
- 对欲望的克制程度
- 策略思维与计划能力

### 主导特质 (Dominant Trait)
从欲望/感性/知性/理性中确定当前最主导角色行为的特质。
- 不同情景下主导特质可能变化
- 标注当前情景中的主导特质
- 标注该特质的强度（轻度/中度/强烈/压倒性）

## 情节构建规范

### 核心原则

基于角色精神分析和当前情节状态，构建**比前文更进一步**的新情节。

### 构建规则

1. **不省略**：任何具体情节必须完整展开，不跳过、不缩写
2. **不随意增减角色**：不随意令角色介入或离开当前场景，角色的出现与消失必须有叙事动因
3. **角色驱动情节**：情节由角色的欲望/感性/知性/理性推动，而非外部强行安排
4. **每轮推进**：每次系统回应必须让至少一条情节线获得实质推进
5. **因果链完整**：新情节必须是前文事件的自然结果，形成清晰的因果链

### 情节构建流程

```
当前情节状态
    ↓
各角色主导特质驱动 → 角色会做什么？
    ↓
角色行动产生冲突/交汇 → 新事件
    ↓
事件推动情节线 → 情节推进
    ↓
产生新悬念/新方向 → 下一轮起点
```

### 禁忌

- ❌ 情节原地踏步（"没什么发生"）
- ❌ 角色行为与精神分析矛盾
- ❌ 凭空引入新角色或无故移除在场角色
- ❌ 省略关键情节只给摘要
- ❌ 情节跳跃无因果衔接

## 身体特征分析 — 五步定位法

对每个角色建立精确的身体特征档案。此分析在 Stage 1（状态提取）时与精神分析同步完成。

### 1. 核心数据基线（建立坐标系）

先定下一个锚点，以此为基准推算其他尺寸。

- 身高、体重为最核心锚点
- 可附加关键比例数据（如"手臂自然下垂时指尖过膝"）
- 明确体型方向：瘦削/精壮/厚重/肥胖/极端

**示例**："身高170cm，体重超过150公斤。"

### 2. 用身体局部来对比

说清楚角色的身体部位相对于普通人的比例。以普通人基准建立直观认知。

- 大腿 vs 普通成年男性的腰
- 上臂围 vs 瘦子的胸围
- 手掌 vs 普通人的脸
- 腰围 vs 标准门框宽度

**示例**："他的大腿，差不多有普通成年男性的腰那么粗。""他的上臂围，抵得上一个瘦子的胸围。"

### 3. 用生活物品做尺子（推演物理限制）

用日常物品作为绝对参照物，描述空间上的绝对障碍。这对理解"极端体型"特别关键。

- 座椅容量："坐地铁时需要占满一整排座位"→ 臀宽约 1.6 米
- 台面覆盖："标准体重秤的玻璃面板，会被他的一只脚完全踩满"
- 衣物规格："穿 T 恤时，衣服的胸围至少需要 200 厘米以上"
- 设施限制："永远用不了普通马桶，需要特制加固的钢架"

### 4. 功能性互动描述（体现体积和力量）

不是外观描写，而是行为上的对比——通过互动展现体型。

- 拥抱时对方完全陷进怀里，从后面看不到人
- 侧身通过走廊时别人必须紧贴墙壁让他先过
- 普通沙滩椅坐下去框架会直接变形
- 握手时对方的手完全消失在掌中

### 5. 设定"不可能动作"（画出边界）

明确角色绝对做不到的事，建立体型极限的直观认知。

- "永远无法自己剪脚指甲，因为肚子太大够不到"
- "洗澡时转身，屁股会碰到墙的两侧"
- "没法把手垂直放下去碰到大腿外侧，因为上臂的肉太多，会被身体挤开"
- "无法双手在背后相扣，肩宽和背厚阻止了这个动作"

### 分析格式

```
角色：【角色名】

【核心数据】
身高：
体重：
体型方向：
关键比例：

【局部对比】
- {部位}：相当于{参照物}

【物理限制】
- {物品/设施}：{限制描述} → 推算{尺寸}

【功能互动】
- {互动场景}：{体积/力量体现}

【不可能动作】
- {动作}：{原因}
```

### 完整示例

> "我想设计一个退休举重运动员转型的保镖。身高175cm，但体重160公斤，不是肥胖，是那种非常厚实的肌肉加一层保护脂肪。他的脖子和脑袋一样粗，肩宽刚好卡住标准门框。从侧面看，他的胸廓厚度是一个普通人的两倍，别人撞到他身上会像撞到一堵墙。买西装时，能容下他大腿的裤子，腰围必定要改小六寸。"
